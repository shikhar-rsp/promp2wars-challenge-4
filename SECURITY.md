# Security Policy

## Reporting a vulnerability
Please open a private security advisory on this repository, or email the
maintainers, rather than filing a public issue. We aim to acknowledge reports
within 72 hours.

## Where security is documented
ATLAS treats security as a first-class concern. The full model — secret
isolation, input validation, prompt-injection defense, CSP/HSTS/CORS, rate
limiting and safe error handling — is documented in
[docs/SECURITY.md](docs/SECURITY.md).

## Supported versions
This is a hackathon submission; `main` is the only supported branch.
